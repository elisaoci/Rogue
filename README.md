# Rogue

Консольное roguelike-приложение на Python с использованием библиотеки curses. Учебный командный проект в рамках School 21.

## Игровой процесс

- 21 уровень подземелья
- 9 комнат на уровень, соединённых коридорами
- 5 типов противников (зомби, вампир, привидение, огр, змей-маг)
- Случайная генерация уровней
- Система предметов (еда, эликсиры, свитки, оружие, сокровища)
- Туман войны
- Сохранение прогресса в JSON

## Управление

| Клавиша | Действие |
|---------|----------|
| WASD | передвижение |
| h | использовать оружие |
| j | использовать еду |
| k | использовать эликсир |
| e | использовать свиток |

## Технологии

- Python 3.10
- Curses

## Архитектура

- Presentation (curses рендеринг)
- Domain (бизнес-логика, генерация уровней)
- Data Layer (сохранение в JSON)
